
# Project Title: World Happiness Data Analysis (2024)

## Overview

This project utilizes a CSV file containing world happiness data from 2024 to explore and understand global happiness trends. The primary purpose of this analysis is:

*   To compare happiness levels in America against the rest of the world.
*   To identify and understand the key factors contributing to happiness within America.

## Important Information about the Data

*   **Data Coverage**: The dataset includes happiness data up until **December 2023**.
*   **Key Metric**: The `Life Ladder` column represents a standardized happiness score assigned to each country.
*   **Source**: This dataset, and similar world happiness data, can be found on **Kaggle**.

## Data Columns (Example - based on typical World Happiness Report data)

While the exact columns might vary, common important columns in such datasets include:

*   `Country name`: Name of the country.
*   `year`: The year the data was collected.
*   `Life Ladder`: The happiness score (dependent variable in analyses).
*   `Log GDP per capita`: Economic output per person.
*   `Social support`: Perceived social support.
*   `Healthy life expectancy at birth`: Health metric.
*   `Freedom to make life choices`: Perceived freedom.
*   `Generosity`: Perceived generosity.
*   `Perceptions of corruption`: Perceived corruption levels.
*   `Positive affect`: Measures of positive emotions.
*   `Negative affect`: Measures of negative emotions.
